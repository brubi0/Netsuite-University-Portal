Quick Reference for Workflow Scheduler Configuration

Workflow Scheduler Overview

  • Runs every 30 minutes as a background process
  • Evaluates scheduled workflows, actions, and transitions
  • Tasks may execute slightly late but never before scheduled time
  • Schedules run in the time zone of the creator

Scheduling Workflows

Requirements

  • Release Status = Released
  • Saved search on same base record type
  • At least one filter in saved search criteria
