üéØ Learning Outcomes

Upon completion of this course, you will be able to:

  • Default data on forms and dynamically change form data during data entry
  • Add field and form-level validations during data entry
  • Control form display based on edit, view, or print mode
  • Send email notifications and automate record creation (tasks, phone calls)
  • Troubleshoot workflow execution using the workflow execution log
  • Generate approval routing workflows with multiple approvers
  • Manipulate sublist line fields on transaction records
