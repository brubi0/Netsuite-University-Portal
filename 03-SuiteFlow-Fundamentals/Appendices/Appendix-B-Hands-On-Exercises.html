Module 0: Course Introduction

Exercise 1: Enabling SuiteFlow

Objective: Verify SuiteFlow feature is enabled in your training account

  1. Navigate to: Setup > Company > Enable Features
  2. Select the SuiteCloud subtab
  3. Locate the SuiteFlow section
  4. Verify that SUITEFLOW is checked
  5. Click Save (if necessary)

Exercise 2: Adjusting User Preferences

Objective: Configure essential user preferences for workflow development

  1. Navigate to: Home > Set Preferences
  2. On the General subtab, configure:
    • Time Zone: Set to your local time zone
    • Date Format: Choose your preferred format
    • Show Internal IDs: ‚úì Check this box (CRITICAL!)
  3. On the Analytics subtab:
    • Check Show List When Only One Result
  4. Click Save

Module 1: SuiteFlow Overview

Exercise 1: Inspect the Custom Furniture Request Record

Objective: Explore the custom record type used throughout the course

  1. Navigate to: Customization > List, Records, & Fields > Record Types
  2. Click on the Furniture Request link
  3. Review the record structure:
    • Standard and custom fields
    • Field types and configurations
    • Required fields and validations

Exercise 2: Create a New Furniture Request Record

Objective: Create a test record to understand the data model

  1. Navigate to: Lists > Custom > Furniture Request > New
  2. Enter test data for all fields
  3. Save the record
  4. Note the record ID and field behaviors

Exercise 3: Create a Simple Workflow

Objective: Build your first workflow with a Show Message action

  1. Navigate to: Customization > Workflow > Workflows > New Workflow
  2. Enter workflow details:
    • Name: "My First Workflow"
    • ID: "wf_my_first"
    • Record Type: Furniture Request
    • Initiate On: Create
    • Release Status: Testing
    • Enable Logging: Checked
  3. Rename default state to "Data Entry"
  4. Add a Show Message action with your welcome message
  5. Test by creating a new Furniture Request record
