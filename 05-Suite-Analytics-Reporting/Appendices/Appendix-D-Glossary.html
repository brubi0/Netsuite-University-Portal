üìñ Appendix D: Glossary

A

Alert Me When Ready

Feature that runs reports in the background and emails notification when complete.

Analytics Data Source

Optimized metadata layer used by Suite Analytics Workbook for faster queries.

Audience

Subtab controlling which roles/users can access a saved search.

Available Filters

Fields users can filter on at runtime without editing the search.

C

Criteria

Filter conditions that determine which records appear in search results.

Custom Label

User-defined column header replacing the default field name.

D

Dashboard View

Deployment option allowing search to appear in Custom Search portlets.

Dataset

Query definition in Suite Analytics Workbook that retrieves data.

Drill-Down

Clicking a value to see underlying detail records.

Dynamic Date

Date filter that adjusts automatically (e.g., "Last Month").

E

Execution Log

Record of who ran a saved search, when, and export actions.

Expression

Logical formula combining criteria with AND/OR/NOT operators.

H

Highlighting

Conditional formatting (colors, bold, images) applied to search results.

J

Joined Field

Field from a related record accessed via ellipsis notation.

K

KPI (Key Performance Indicator)

Quantifiable metric measuring business performance.

KPI Meter

Dashboard portlet showing a gauge-style visualization.

M

Main Line

Header-level row of a transaction (vs. line items).

Mine

Variable filtering results to logged-in user's records.

P

Pivot Table

Aggregated data view with rows, columns, and summary values.

Portlet

Dashboard component displaying specific content or data.

Public

Checkbox making a saved search visible to all users.

R

Reminders

Dashboard portlet showing counts from saved searches.

Results

Subtab defining which columns appear in search output.

S

Saved Search

Reusable query returning filtered, formatted record data.

Scorecard

KPI portlet comparing metrics across date ranges.

Summary Type

Aggregation function: Sum, Count, Average, Min, Max, Group.

T

Trend Graph

KPI portlet showing line chart of values over time.

W

Workbook

Suite Analytics container holding visualizations linked to datasets.
