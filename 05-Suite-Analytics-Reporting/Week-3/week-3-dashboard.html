Week Overview

This week takes your saved search skills to the next level. You'll learn to filter by the logged-in user, combine criteria with AND/OR/NOT logic, highlight results with colors and images, configure deployment options, and set up email notifications. These advanced techniques let you build searches that are both powerful and user-friendly.

Learning Objectives

  • Use the "Mine" variable to filter by logged-in user
  • Combine criteria with AND, OR, NOT logic and parentheses
  • Apply conditional highlighting (colors, bold, images)
  • Configure deployment options for lists, dashboards, and menus
  • Set up notification emails triggered by record changes
  • Create scheduled email distributions
