Module Overview

Report tools help you navigate and adjust the display of information in NetSuite reports. This module covers the Options menu, graph visualization, and expand/collapse controls for viewing data at different levels of detail.

The Options Menu

Click the Options icon (wrench/gear icon) to access display settings for the report.

Option Description
View: Report Plane Standard report view without grid lines
View: Report Grid Adds horizontal and vertical grid lines ‚Äî useful for aligning data in multi-column reports
Expand Level Choose the default level to which subgroup details should appear
Display Title Show or hide the report title heading
Show Drill Down Balloons Show hover indicators that explain what happens when you click (enabled by default)

üí° Example: Enable Grid View

When viewing a report with multiple location columns, the data can be hard to read. Enable Report Grid view to add visual guides that help you line up values across columns.

Graph Visualization

Click the Graph icon (chart icon) to display report data as a visual graph.

üìä Graph Settings

  • Graph Type ‚Äî Bar, line, pie, area charts
  • X-Axis ‚Äî Select the dimension for horizontal axis
  • Y-Axis ‚Äî Select the measure for vertical axis
  • Top Values ‚Äî Limit to top N values for cleaner visualization

üí° Pro Tip

Graphs are great for presentations and quick visual analysis. However, for detailed data work, stick with the tabular view where you can drill down into individual transactions.

Expand & Collapse Controls

Reports with grouped data include up and down arrows to control the level of detail displayed.

‚¨ÜÔ∏è Collapse (Up Arrow)

Collapse all groups to see only top-level totals. Makes it easier to see the overall picture and grand totals per group.

‚¨áÔ∏è Expand (Down Arrow)

Expand groups to the next level. Click repeatedly to expand level by level until you see all subgroups and subtotals.

üí° Example: Quick Overview

On a Sales by Customer report with many customers:

  1. Click the Up Arrow to collapse all customer groups
  2. See only the grand total for each customer
  3. Click the Down Arrow to expand and see subtotals by item type
  4. Continue expanding to see individual line items

üéØ Key Takeaways

  • Options menu (wrench icon) controls display settings like grid view and expand level
  • Report Grid view adds lines to help align multi-column data
  • Graph icon converts report data to visual charts
  • Up/Down arrows collapse or expand grouped data
  • Drill-down balloons (hover hints) can be toggled on/off in Options
