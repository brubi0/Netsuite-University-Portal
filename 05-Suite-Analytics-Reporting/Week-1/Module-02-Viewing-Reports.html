Module Overview

NetSuite reports display transaction information in real time. You can view summary data, drill down to details, navigate to individual transaction records, and search within reports. Understanding these navigation features helps you quickly find the data you need.

Summary vs. Detail Views

Most NetSuite reports offer two ways to view data: summary (aggregated totals) and detail (individual transactions).

üìä Summary View

Displays aggregated totals grouped by dimension (customer, item, date, etc.)

  • High-level overview of data
  • Totals and subtotals by group
  • Click amounts to drill down
  • Best for quick analysis

üìã Detail View

Shows individual transactions that make up the aggregated totals

  • Line-by-line transaction list
  • Click to open individual records
  • Research and audit trail
  • Best for investigating specifics

Drill-Down Navigation

üîó View Detail Link

From the summary view, click View Detail to see all underlying transactions that make up the aggregated total.

  1. In Summary View, locate the total you want to investigate
  2. Click View Detail to see all transactions
  3. Click Back to Summary to return to the overview

üí∞ Click Amount Values

Click directly on an amount in the summary view to see only the transactions for that specific group.

  1. In Summary View, click on a specific amount (e.g., $15,000 for Customer A)
  2. The detail view shows only transactions from that single group
  3. This is more targeted than "View Detail" which shows all transactions

üí° Example: Sales by Customer Report

You want to review this year's sales transactions for a specific customer.
  1. Navigate to the Sales by Customer report
  2. In the summary view, locate your customer in the list
  3. Click the amount next to that customer's name
  4. The detail view shows all sales transactions for that customer only
  5. Click any transaction to open the full record (invoice, sales order, etc.)
  6. Click Back to return to the report

Using the Find Field

üîç Search Within Reports

The Find field lets you search for specific text within a report without leaving the page.

  1. Enter a search term in the Find field
  2. Press Tab or Enter to activate the filter
  3. Matching text is highlighted in the report
  4. Red markers appear in the navigation pane showing where matches are found
  5. Click a marker to jump to that section of the report

üí° Pro Tip

The Find field is especially helpful for navigating large reports. Instead of scrolling through hundreds of rows, search for the customer, item, or value you need and jump directly to it using the navigation markers.

Drill-Down Balloons

üí¨ Hover Indicators

When you hover over clickable fields in a report, drill-down balloons appear to indicate what will open when you click.

  • Balloons tell you if clicking opens a detail view or a specific record
  • This feature is enabled by default
  • Can be disabled in Report Options if preferred

Opening Individual Records

üìÑ From Detail View to Transaction

The detail view provides a convenient way to navigate to and research related transactions.

  1. In the detail view, locate the transaction you want to open
  2. Click the transaction number or linked field
  3. The full transaction record opens
  4. Click Back in your browser to return to the report

üéØ Key Takeaways

  • Summary View shows aggregated totals; Detail View shows individual transactions
  • Click amounts to drill down to transactions for a specific group
  • Use the Find field to search and navigate large reports quickly
  • Drill-down balloons indicate what opens when you click
  • From detail view, click transactions to open the full record
  • Expand/Collapse controls let you adjust the level of detail displayed
