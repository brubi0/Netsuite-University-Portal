What are KPIs?

Key Performance Indicators (KPIs) are quantifiable metrics that measure business performance. NetSuite provides 75+ prepackaged KPIs based on standard reports, displayed through various dashboard portlets. KPIs let executives and managers monitor critical metrics at a glance.

KPI Portlet Types

üìä Key Performance Indicators

Summary lines showing current values with drill-down to underlying data.

  • Multiple KPIs in one portlet
  • Click values to drill down
  • Shows trend indicators

üéØ KPI Meter

Semi-circular gauge showing a single value against thresholds.

  • Visual speedometer style
  • Compare to target values
  • Color-coded ranges

üìâ Trend Graph

Line chart showing KPI values over time.

  • Multiple KPIs on one chart
  • Select time intervals
  • Track trends visually

üìã KPI Scorecard

Tabular comparison of KPIs across date ranges.

  • Compare periods side-by-side
  • Highlight variances
  • Complex multi-KPI views
