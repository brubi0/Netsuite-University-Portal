Week Overview

This final week covers NetSuite's executive-level analytics tools. You'll learn to display Key Performance Indicators (KPIs) on dashboards using various portlet types, create custom KPIs from saved searches, and build interactive data visualizations with Suite Analytics Workbook. These tools transform raw data into actionable business intelligence.

Learning Objectives

  • Add standard KPIs to dashboards using multiple portlet types
  • Create custom KPIs from saved searches with summary fields
  • Build KPI Meters, Trend Graphs, and Scorecards
  • Understand Suite Analytics Workbook architecture (datasets + workbooks)
  • Create visualizations: tables, pivot tables, and charts
  • Share workbooks and display them on dashboards via Analytics Portlet
