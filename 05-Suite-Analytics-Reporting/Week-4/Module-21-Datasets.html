What is a Dataset?

A dataset is a query that retrieves data from NetSuite's Analytics Data Source. Think of it like a saved search, but optimized for workbook visualizations. Datasets define what data to retrieve; workbooks define how to display it.

Creating a Dataset

üìç Steps

  1. Navigate to Analytics ‚Üí Datasets
  2. Click New Dataset
  3. Select a record type (Transaction, Customer, Item, etc.)
  4. Drag fields from the left panel to the Columns area
  5. Add Criteria (filters) to limit results
  6. Click Save

Dataset Features

  • Drag-and-drop field selection
  • Linked fields ‚Äî access related record data
  • Formula fields ‚Äî create calculated columns
  • Criteria ‚Äî filter rows returned
  • Preview ‚Äî see sample data before saving

üí° Example: Sales Transaction Dataset

  • Record Type: Transaction
  • Fields: Date, Customer, Item, Quantity, Amount
  • Criteria: Type = Invoice, Main Line = False

üéØ Key Takeaways
